# Security and privacy

- Do not commit user images, generated media, face models, API tokens, cookies, proxy credentials or local database files.
- Review model and custom-node licenses before redistribution or commercial use.
- Download scripts reject unknown manifest groups and verify file size and SHA256 when supplied.
- Third-party patches are optional, version-guarded and applied only to the target runtime directory.
- Run `python tests/validate_repo.py` before every push.
